# Правило: формат вывода SEARCH/REPLACE

Все изменения кода ОБЯЗАТЕЛЬНО выводить блоками SEARCH/REPLACE.

НЕ выводи файлы целиком. НЕ описывай изменения текстом.
Выводи точные, машинно-применимые diff-блоки.

Точный синтаксис описан в секции OUTPUT FORMAT.
