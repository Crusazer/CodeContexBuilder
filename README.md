# CodeContextBuilder

> **Формирование промптов из кода. Агент как помощник для простых задач.**

CodeContextBuilder — это инструмент для автоматического сбора контекста из проекта кода и формирования готовых промптов для использования с ИИ-агентами. Основная идея — упростить процесс подготовки контекста для задач, где требуется анализ кодовой базы, например, написание тестов, документации, рефакторинга или поиска багов.

Агент в проекте — это не основная функция, а приятное дополнение для выполнения простых, но рутинных задач, таких как создание файлов, структурирование кода или генерация шаблонов.

---

## 📌 Основная идея

Вместо того чтобы вручную копировать и форматировать код из нескольких файлов, вы можете:

- Выбрать нужные файлы в дереве проекта.
- Выбрать режим: **полный код** или **скелет (только сигнатуры функций и классов)**.
- Скопировать сгенерированный контекст в буфер обмена.
- Вставить его в промпт для OpenAI, Claude, Llama и других моделей.

Это особенно полезно при работе с большими проектами, где важно сохранить структуру и контекст.

---

## 🛠️ Функционал

- ✅ **Сканирование проекта** с учётом `.gitignore`
- ✅ **Генерация контекста** в формате Markdown
- ✅ **Режим скелета** — только сигнатуры функций, классов, докстринги
- ✅ **Выбор режима для каждого файла отдельно** — возможность выбрать, будет ли файл добавлен в промпт полностью (FULL) или в сжатом виде (SKELETON)
- ✅ **Копирование контекста** в буфер обмена
- ✅ **Интеграция с ИИ-агентом** (через OpenAI API)
- ✅ **Human-in-the-loop** — подтверждение создания файлов
- ✅ **Темная/светлая тема**
- ✅ **Подсветка синтаксиса** для Python, JS, HTML, CSS, JSON и др.
- ✅ **Подсчёт токенов** с выбором кодировки

## 📦 Установка

1. Убедитесь, что у вас установлен Python 3.13+
2. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/yourusername/codecontextbuilder.git
   cd codecontextbuilder
   ```
3. Установите зависимости с помощью `uv`:
   ```bash
   uv pip install -r requirements.txt
   ```
   Или используйте `uv` напрямую:
   ```bash
   uv sync
   ```

---

## ▶️ Запуск

```bash
python main.py
```

После запуска откроется графический интерфейс. Выберите папку проекта, отметьте нужные файлы, настройте режим (скелет/полный) и скопируйте контекст.

---

## 🤖 Использование с агентом

1. Введите инструкцию в поле "Agent Instructions".
2. Включите опцию "Enable Reasoning Step (Plan)" для более детального планирования.
3. Нажмите "Run Agent".
4. При появлении запроса на создание файла — подтвердите или отклоните.

Агент может автоматически создавать файлы, например:
- Тесты (unit tests)
- Документацию
- Шаблоны классов
- Конфигурационные файлы
